# 📌 Project Overview

This project performs an A/B Testing Comparison between Facebook Ads and Google AdWords advertising campaigns using Python.

The objective is to compare both advertising platforms based on their performance metrics such as ad views, clicks, conversions, conversion rate, cost per click (CPC), and overall campaign effectiveness.

The project includes Exploratory Data Analysis (EDA), Statistical Hypothesis Testing, Correlation Analysis, Linear Regression, Time-Series Analysis, and Business Insights.

---

# 🎯 Business Problem

Marketing agencies invest significant budgets on online advertisements. Choosing the right advertising platform is essential for maximizing Return on Investment (ROI).

This project identifies whether Facebook Ads or Google AdWords provides better marketing performance by analyzing campaign data.

---

# ❓ Research Question

Which advertising platform performs better in terms of:

- Clicks
- Conversions
- Cost Effectiveness
- Conversion Rate
- Cost Per Click (CPC)

# 📌 Key Findings

- Facebook and AdWords show similar conversion performance.
- Clicks positively influence conversions.
- Regression model predicts conversions based on ad clicks.
- Monthly conversion trends indicate seasonal fluctuations.
- September recorded the lowest Cost Per Conversion.
- Statistical testing indicates that neither platform significantly outperforms the other.

---

# 📌 Conclusion

Both Facebook Ads and Google AdWords are effective advertising platforms.

The statistical analysis shows no significant difference between their average conversions. Businesses should optimize campaigns based on budget, audience targeting, and campaign objectives instead of relying on a single platform.
